# Exercise: Setup

In your CS50 IDE, create a new directory for the Web Track:

```nohighlight
$ cd ~/workspace
$ mkdir web-track
```

Inside the new directory, make some subfolders:

```nohighlight
$ cd web-track
$ mkdir studios
$ mkdir exercises
```

...And that's it for now!
