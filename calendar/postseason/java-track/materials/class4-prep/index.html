Class 4 Prep

Before coming to Class 4, please complete the following tasks:

Inheritance

Task Resource Type Link Instructions
Watch Follow Along Laptops and Tablets Follow along with Doug as he explains inheritance while creating different types of computers.
Read Resource Java Documentation on Inheritance The official Java documentation on inheritance is a useful reference.
Read Resource Polymorphism A great explanation of Polymorphism and why it is useful.
Read Resource Inheritance vs. Composition A great blog post that tries to illustrate when you should use composition or inheritance to solve a problem.
Read Resource Access Modifiers More information about access modifiers in Java. Pay special attention to the protected keyword.
Do Exercise Robotic Diversification Robots are good for many different things. When you complete this exercise you will be able to create robots that can perform just about any activity.

GUIs

Task Resource Type Link Instructions
Watch Follow Along Creating GUIs Create a GUI for the Computer class with Doug.
Read Resource GUI Tutorial This is a very detailed tutorial that explains the different components of a GUI as well as how to create them.
Read Resource Swing Documentation This set of tutorials will be a great reference when you start building your own GUIs.
Read Resource Windowbuilder Documentation This documenation is for the Windowbuilder GUI designer in Eclipse. Probably not something you want to read all at once, but a great reference to use while you're coding.
Read Resource Principles of Good Interface Design A great resource for things you should consider when designing and implementing your user interface.
Do Exercise Robotic Interaction Create a fancy user interface for interacting with your robots

Now you're ready for Class 4
