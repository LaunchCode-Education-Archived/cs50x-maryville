Read Lecture Notes Week 4 / Pointers These notes reveal how, under the hood, a string is just a pointer to a char. This means that the "value" inside a string variable is not actually the contents of the text. Instead it is a number pointing to the memory address where the text is truly located-- more specifically, where the first char is located.
